# Infrastructure as Code (IaC) Bakeoff : ARM/Bicep vs Terraform vs Pulumi

What's the BEST infrastructure as code (IaC) for me? Here's the consultant's answer...It depends. What's your target and what are your priorities? Where do you need to work? How do you do DevOps? All these and more feed into your decision, and in today's presentation we'll explore the options for workiing with Azure.
